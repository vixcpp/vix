# Vix Requests Examples

These umbrella examples mirror common Python `requests` workflows: query params, headers, JSON/form POSTs, sessions, auth headers, downloads, retries, async calls, pagination, and a small API wrapper.

Use `VIX_REQUESTS_BASE_URL` or the per-example environment variables to point examples at a local mock service, an internal API, or a public HTTP/HTTPS test endpoint.

Default base URL: `https://httpbin.org`
